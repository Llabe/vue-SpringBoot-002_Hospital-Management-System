# SpringBoot-Vue-067基于SpringBoot的中小型医院网站

#### Description
基于 Spring Boot 2.7+ 和 MyBatis Plus 的医院管理系统，整合 Shiro 权限框架与 MySQL 数据库，实现患者在线预约挂号、医师诊疗管理、药品库存监控等核心功能，采用前后端分离架构提升系统可维护性。
